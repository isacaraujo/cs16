INFORMACOES SOBRE A VERSAO 1.3
------------------------------
A versao 1.3 eh TOTALMENTE COMPATIVEL com as versoes anteriores (1.0, 1.1 e 1.2),
sendo que APENAS O SERVIDOR precisa estar com a versao 1.2 para um jogo com 32
jogadores. Ou seja, NINGUEM precisa atualizar o seu cs_rio para jogar em servidores
de 32 jogadores, basta o servidor estar atualizado e rodando com 32 jogadores.
Os jogadores apenas precisas fazer o upgrade para poder ver o mapa no novo modo
espectador introduzino no CS 1.4.
As diferencas entre a versao 1.3 e 1.0 sao:
- Suporta 32 jogadores
- Bug do refem no CS 1.5 corrigido
- Novos waypoints compativeis com o PODBot 2.5
- Overviews para o HLTV
- As balas perdidas estao funcionando como deveriam
- Easter egg diferente.

VERSION 1.3 INFORMATION
-----------------------
The version 1.3 is TOTALLY COMPATIBLE with all older version (1.0, 1.1 and 1.2).
ONLY THE SERVER needs to be updated for a 32 players game. The players DON'T NEED
TO UPDATE their cs_rio to join a 32-player server, they can connect with the original
cs_rio. The players just need to upgrade to be able to watch the map in the new
spectator mode introduced on CS 1.4.
The diferences between version 1.2 and 1.0 are:
- 32 player spawns
- Hostage bug on CS 1.5 fixed
- Stray bullets now work as they should
- New waypoints compatible with PODBot 2.5
- HLTV overviews included
- New Easter Egg
